# Microsoft 365 Help Desk Homelab

## Overview
This project simulates real-world help desk scenarios using Microsoft 365.  
The goal is to practice user support, troubleshooting, and ticket documentation.

## Skills Demonstrated
- User account management
- Password resets and account recovery
- Microsoft Outlook troubleshooting
- MFA (Multi-Factor Authentication) setup
- License management
- Basic admin troubleshooting

## Tools Used
- Microsoft 365 Admin Center
- Microsoft Entra ID
- Microsoft Outlook

## Lab Setup
See /setup/lab-setup.md for full environment configuration.

## Ticket Scenarios
- Password Reset
- Outlook Not Syncing
- MFA Issues
- License Removal Impact

Each ticket includes:
- Problem
- Cause
- Troubleshooting steps
- Resolution
