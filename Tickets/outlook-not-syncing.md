# Ticket: Outlook Not Syncing

## Issue
User email not updating in Outlook.

## Cause
Corrupted Outlook profile.

## Troubleshooting Steps
1. Signed out of Outlook
2. Removed existing profile
3. Re-added account
4. Restarted Outlook

## Resolution
Email sync restored successfully.

- Outlook troubleshooting
- Client-side issue resolution
