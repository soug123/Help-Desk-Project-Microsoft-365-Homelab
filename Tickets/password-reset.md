# Ticket: Password Reset

## Issue
User unable to log into Microsoft 365 account.

![Login Failure](../screenshots/password-reset-1-login-fail.png)

## Cause
User forgot password.

## Troubleshooting Steps
1. Accessed Microsoft 365 Admin Center  
![User Selected](../screenshots/password-reset-2-user-selected.png)

2. Initiated password reset  
![Reset Action](../screenshots/password-reset-3-reset-action.png)

3. Generated temporary password  
![Temp Password](../screenshots/password-reset-4-temp-password.png)

## Resolution
User logged in successfully and updated password.
